Repeating conversations, Can easily be replaced with AI devices in future. 

One Such example that I tried replicating is 'Drive Thru System' using Alexa, This project was developed back in 2017 as a demo.
Drive Thru is one conversation that is exactly the same throughout the day, Menu items might be different but the overall conversation is consistent on customer order.

In this Project I have created a Drive Thru system Automation based on publicly available Wendys menu.
With Automation of Drive thru menu, Many of the Human challenges can be overcome improving Customer Satisfaction.

This Project is developed in Alexa, but we can develop a custom Program as well from scratch using NLP and publicly available API's.
